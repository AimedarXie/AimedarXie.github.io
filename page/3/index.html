《被认识十六年的男生PUA的惨痛经历//Update近期没有更新的原因// 求求上帝对我好一点吧》

我知道这样有些冒犯，不过作为一个up，我想受到这种程度的分析还是可以被原谅的吧。

有几个点是很有趣的：

  1. 我买了ps5键盘耳环项链所有他会喜欢的东西/他就送我一个娃娃/我好亏啊，他甚至都没有喜欢过我。
    特别看重某种消费符号的相互赠予。
    而且赠予的符号与个人认知是有某种关联的：相符代表尊重和肯定，不相符则被认为是蔑视。
    亏这个词特别有味道。

  2. 因为我爱他，我五年写了167（大概吧）篇小作文/他都没有爱过我，精力金钱都没有。
    这种“爱”带有很强烈的个人主义的味道。
    首先，对于喜欢的判定是由自己掌控的。我可以判断我喜欢他，同时也可以否认他喜欢我。
    其次，暗恋和小作文这种形式充满了自我满足，或者倒错的味道：爱伴随着痛苦和误解，那我承受误解和痛苦会得到爱的回报。至于会不会对爱的对象产生影响则无关紧要。

  3. 发朋友圈/做视频哭诉
    非常需要他人的目光来确立自身。

当然我毫无批判的想法，每个人都有构建自己的爱欲结构的权力，这应该是平等的。我没能力也没必要去影响他人。但是up的爱欲结构给自己带来痛苦（当然这种痛苦也有可能让她很爽），我觉得了解这种爱欲结构还是对于我了解别人很有帮助的。从这个维度上我很尊敬和感激她的坦诚。
不过后来小两口和好了，她还在继续更新，我觉得我还可以再研究一会儿。

致富_3

发表于 2021-02-28 更新于 2021-03-01

旧友泻药同志深夜来电，起手式就是关于时间性的哲学讨论。我又惊又喜，正想着我该怎么装作对这问题有所思考；结果话题一转，他告诉我，昨天我和他和他妈去余杭的在建楼盘看房，今天上午就签了购房合同。我松了一口气，又倒吸了一口气，一进一出就显得很平淡自如。

他有自己的焦虑，我也有我的——但终归逃不出年轻人对未来的迷茫。他觉得每个月的分期还款让他突然感受到了生存的代价，觉得从无忧无虑的校园幻梦中惊醒，但又不知道该怎么生存。我建议他多问问学长，仔细看看别人的生活，喜欢就有样学样，不喜欢就溜之大吉。我他妈的哪儿有资格说这话。

房子的思考，以及关于这一个十年 年轻人的焦虑。

  1. 有人总是幻想存在一个过去的维度，怀念人们充满革命热情不需要为生计发愁，为房贷焦虑。BS，说这种话的人一般都是自己先背弃了革命梦想的人。首先，从古至今，大多数人需要为了生物层面的生存而努力；我也不觉得现在条件达到了，无论是社会还是个人层面，我们不需要这种的付出。其次，回避思考自己生产生活的方式就是回避自己的主体性，you have to pay。

  2. 关于为了生存资料而打工的压力是大还是小，代价是高还是低，我觉得只有自己说了算。父母那一代人既是既得利益者，又是天使投资人，外界的观点太复杂又带有时代局限。而且我觉得有人觉得生存压力大是因为他自己让渡出了其他的精神空间，只要你不去反思，或者贪恋着capitalism为你编织的便利，那这代价就是你要支付的。

现在来看，房子的属性包括但不限于：生存资料|生活资格（类似户口）|生活方式|投资品

  1. 当然，我们做事也要讲方法。房产更是有投资属性的。泻药妈妈绝对是为了投资。hz能否维持房价？我觉得人口下行的压力还要10年左右才会体现，而且南方没那么快。改善型住房的需求也还存在。但是我不看好未来，因为最近hz经济靠阿里的一众互联网企业拉动，工业我觉得是日渐式微了。这个泡沫不知道什么时候会破。我只知道人口下行压力对现在的社会运行逻辑绝对有毁灭性的挑战，生产和社保绝对会有巨大变革，杭州还不配和东京落得一个下场。

  2. md。我也想当包租婆，炒个房，自己还没什么建树就当既得利益者。md我配吗？现在炒房，或者整个发展的逻辑就是透支未来，代表就是借贷这个模式。只要这个未来没被透支完，泡沫就能维持。大家辛苦提升的生产效率都给了手机，移动互联网，金融。但是要破局，或者勉强维持，除了提升生产效率别无他法。

  3. 我的话，40岁前能有个地方落脚就算是成功了吧笑。真是充满幻想的杂谈，自己明明还在啃老。所以欢迎包租婆，我的联系方式是…jokes，祝大家都能成为（或者找到）包租婆。

发表于 2021-02-16 更新于 2021-03-06

Aimedar删了狮子的好友。

关系可能要告一段落了。（2020.8.16-2021.1.23）

我要根据未来，回溯性的构建这段关系。

首先指出这段关系中吊诡的地方：

  1. 始于异国。语境有一定的差距。可见的未来会长期异国。

  2. 最近三年基本没有社交网络之外的接触，对于对方的叙事基本一无所知。我不知道吸引力是怎么维持的。

  3. 直到最后也没有建立有效的相处方式。基本就是聊一些学习、生活和知乎话题。爱好有一定的偏差，手游、书法、国创、小说、剑侠我基本上都不玩。

我要指出一些基本的冲突，也可以理解为我的不满，所以相当主观：

  1. 大的人生方向上存在分歧。

    异国是一个很有挑战性的背景。我不知道要怎么把对方编织进我的叙事里，语境一直是割裂的。精神上这是一个一直存在的负担。

  2. 对于对方生活中具体的挑战基本都是无力的。

    我最不能接受的是这个。缺少一个实践的维度，交往很空洞，基本上脱离了个人的发展。由此，我最大的动机，对于互相成就的渴望就一直被压抑。

    而且失去了这个维度，我不知道在何种意义上我们对彼此应当负责。面对一些负面情绪我当然可以去做一些所谓的安慰，但是这根本不解决实际问题。我不想一直做一个树洞，或者某种旁观者，但是我没有找到合适的参与进对方生活的角度。这让我对自己很失望，没有价值感。

  3. 交互模式比较初级。

    交互模式永远存在改进的空间。这是比较术的东西，也比较私人，我不是很想在这里讨论。但是我的这种态度可能也需要反思。我之前一直轻视认识论的维度，如何准确的传达自己的想法，如何把握别人释放的信号，如何调动别人的情绪，我应该对这些问题提高重视。不同情绪下，你以为是某种本体论的维度，实际上是认识论的维度。

  4. 态度

    我有一些地方做得不是很好，比如有一两次拒绝过一起玩的请求，这样想来可能当时对方是怀有很大期待的。但是我还是不太能接受对方的态度，少有那种自觉的建设性的表现，我至少希望能被同样的对待。但我感受到的不是钝化，而是回避。

人间错题集_1

发表于 2020-12-01 更新于 2021-04-07

狮子：我觉得我不被需要。我觉得你对我一无所求。

Aimedar：一个小视频：https://www.bilibili.com/video/BV1hb4115724
一段童年阴影故事：讲述自己小学时是一个怎样的鸡蛋。如何利用别人的好感而不知，以及报复性的自我厌恶。

解析：

表面上看狮子想传达的是一种自我否定。那标准答案应该是及时的对自我否定做出否定，like你对我很重要，我需要你类似的。

如果这么想，我认为这是一种标准错解。首先这不会起到安慰的效果，因为不存在沟通的语境。肯定的语境下问不出这样的问题，在一个充满自我怀疑的语境下的安慰也可能被理解为敷衍。其次我认为自我否定的理解相当片面——正确的角度应该着眼于双方之间的关系。

从权力的角度，“我觉得我不被需要”应该被理解为一种权力的不对等。自己在沟通中处于弱势一方，因此感受到压抑。那么这样的回答是否可以接受？“我也经常怀疑我是否被您需要，不知道哪一天您会离我而去。我只知道，您对我很重要。”从关系的本质上入手，首先是平等性，构建一个平等的对话的语境。然后在这个语境上再做出肯定。同样的思路，也可以先进行更深刻的自我否定，证明自己的无足轻重然后再来沟通。

但是我觉得这还是不够的。更深的不安来自被看见的需求。
“我觉得你不需要我”。我以前在狮子心里种下了一个种子，“我不太分的清喜欢您和我的绿萝的区别”。反抗被视作对象而非主体的压抑，希望实现穿越公式化处理被注视的需求。我不知道该怎么处理这种需求，而且我认为这种需求深埋于我们每个人心中；或许这就是为什么我们追求与他人的沟通，哪怕面前是地狱。

就写到这里吧。我有点累了。
