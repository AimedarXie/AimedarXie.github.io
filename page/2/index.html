人间错题集_2

发表于 2021-02-16 更新于 2021-03-06

Aimedar删了狮子的好友。

关系可能要告一段落了。（2020.8.16-2021.1.23）

我要根据未来，回溯性的构建这段关系。

首先指出这段关系中吊诡的地方：

  1. 始于异国。语境有一定的差距。可见的未来会长期异国。

  2. 最近三年基本没有社交网络之外的接触，对于对方的叙事基本一无所知。我不知道吸引力是怎么维持的。

  3. 直到最后也没有建立有效的相处方式。基本就是聊一些学习、生活和知乎话题。爱好有一定的偏差，手游、书法、国创、小说、剑侠我基本上都不玩。

我要指出一些基本的冲突，也可以理解为我的不满，所以相当主观：

  1. 大的人生方向上存在分歧。

    异国是一个很有挑战性的背景。我不知道要怎么把对方编织进我的叙事里，语境一直是割裂的。精神上这是一个一直存在的负担。

  2. 对于对方生活中具体的挑战基本都是无力的。

    我最不能接受的是这个。缺少一个实践的维度，交往很空洞，基本上脱离了个人的发展。由此，我最大的动机，对于互相成就的渴望就一直被压抑。

    而且失去了这个维度，我不知道在何种意义上我们对彼此应当负责。面对一些负面情绪我当然可以去做一些所谓的安慰，但是这根本不解决实际问题。我不想一直做一个树洞，或者某种旁观者，但是我没有找到合适的参与进对方生活的角度。这让我对自己很失望，没有价值感。

  3. 交互模式比较初级。

    交互模式永远存在改进的空间。这是比较术的东西，也比较私人，我不是很想在这里讨论。但是我的这种态度可能也需要反思。我之前一直轻视认识论的维度，如何准确的传达自己的想法，如何把握别人释放的信号，如何调动别人的情绪，我应该对这些问题提高重视。不同情绪下，你以为是某种本体论的维度，实际上是认识论的维度。

  4. 态度

    我有一些地方做得不是很好，比如有一两次拒绝过一起玩的请求，这样想来可能当时对方是怀有很大期待的。但是我还是不太能接受对方的态度，少有那种自觉的建设性的表现，我至少希望能被同样的对待。但我感受到的不是钝化，而是回避。

反思的反思

我没什么分析的工具，关于冲突的梳理受到了未明子主义主义
https://www.bilibili.com/video/av928763237
的影响，大致上想设计 场域论-本体论-认识论-目的论 的结构，但不严谨。

人间错题集_1

发表于 2020-12-01 更新于 2021-03-05

狮子：我觉得我不被需要。我觉得你对我一无所求。

Aimedar：一个小视频：https://www.bilibili.com/video/BV1hb4115724
一段童年阴影故事：讲述自己小学时是一个怎样的鸡蛋。如何利用别人的好感而不知，以及报复性的自我厌恶。

解析：

表面上看狮子想传达的是一种自我否定。那标准答案应该是及时的对自我否定做出否定，like你对我很重要，我需要你类似的。

如果这么想，我认为这是一种标准错解。首先这不会起到安慰的效果，因为不存在沟通的语境。肯定的语境下问不出这样的问题，在一个充满自我怀疑的语境下的安慰也可能被理解为敷衍。其次我认为自我否定的理解相当片面——正确的角度应该着眼于双方之间的关系。

从权力的角度，“我觉得我不被需要”应该被理解为一种权力的不对等。自己在沟通中处于弱势一方，因此感受到压抑。那么这样的回答是否可以接受？“我也经常怀疑我是否被您需要，不知道哪一天您会离我而去。我只知道，您对我很重要。”从关系的本质上入手，首先是平等性，构建一个平等的对话的语境。然后在这个语境上再做出肯定。同样的思路，也可以先进行更深刻的自我否定，证明自己的无足轻重然后再来沟通。

但是我觉得这还是不够的。更深的不安来自被看见的需求。
“我觉得你不需要我”。我以前在狮子心里种下了一个种子，“我不太分的清喜欢您和我的绿萝的区别”。反抗被视作对象而非主体的压抑，希望实现穿越公式化处理被注视的需求。我不知道该怎么处理这种需求，而且我认为这种需求深埋于我们每个人心中；或许这就是为什么我们追求与他人的沟通，哪怕面前是地狱。

就写到这里吧。我有点累了。

https://www.bilibili.com/video/BV1ny4y167eY?t=1556

建站日志_2

发表于 2020-04-30

启发我建站的博客是《小黑屋》，这也是一个个人博客。我猜测博主是研二的同学，物化相关的实验室。我看到他做了一些学习整理，虽然毫无学术意义，但是至少是一种个人总结与输出。我很受启发。

我建立blog的初衷是也想做一些个人的整理与输出。

输出对象：

我想这个站应该是给陌生人看的。我想把这里作为我私人社交圈与社会公共环境的边界，至少是我设计的边界。看我发布的东西，不需要认识我是谁，倒不如说这是我设计的，留给陌生人进入我的世界的一扇门。

输出内容：

我希望做一些整理性，工具性的内容。学习知识的总结是我首要考虑的内容，因为一方面这对我个人而言是一个自我梳理和知识系统建立的机会；另一方面，看这样的内容，社交门槛低的同时收益高，很适合陌生人来浏览，浏览之后黏性也强（吗？）。

同时我也希望能做一些个人化的内容。然而个人化的尺寸很难把握。我不想把我的博客变成一个自我宣泄的垃圾场。不要日记。随笔也要注意尺度。

我一天看十个视频，吃三餐饭，但是这种记忆忘得很快，而且不要也罢。我想做一些值得记忆的东西，有价值的东西。它会记录我的脚步，一点一滴的塑造我这个人，同时通过这种回顾我也将找到接下来的道路。

什么是值得记忆的，有超越时空的价值的内容？
