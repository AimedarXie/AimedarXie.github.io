人间错题集_11

发表于 2021-06-20 更新于 2021-06-23

吸取经验教训

​ 6.7号答辩结束，拿了优秀答辩。之后端午节爸妈来北京，三天去西山的古道，南锣鼓巷和陶然亭公园走了一走。毕设和父母的精神压力减轻了，现在有余力总结一下上半年遭遇的挫折与困境，以及规划未来的行动。（写到这里花了我两天，看来我还需要训练）

如果现在让我指导半年前的自己做毕设，有几点我会对自己强调。
  1. 毕业设计的唯一目的是完成毕业设计。什么学术价值与旨趣，什么科学方法的训练，不过是毕业设计的副产物。这就是说，别总想着脱离实践，先设立一个目标，培养一个虚空中的学术能力，然后再来开展工作；很容易被这种虚头八脑的东西困住，止步不前甚至放弃，因为事情根本不是这么做的。唯一要考虑的事情是如何完成毕业设计，找一些好用的方法和工具，提高自己的效率。尽量避免价值判断，在完成前是没有资格挑挑拣拣的，完成后也没有资格自我评价学术成果。
  2. 要有计划。计划失败也是一种计划，但是不能不计代价的去追寻想象中的成功。最重要的资源是时间，其他资源也可以折合成时间，所以可以按照时间来设定计划。特别是要和别人合作时，因为别人也要做计划，因此不能只考虑时间长度，而要给出一个时间轴上的节点。
  3. 如何应对未知的挑战。总会遇到超出自己知识和能力的情况。首先要破除全能幻想，不是所有挑战都是可以自己独立完成的。卡住了就多尝试，没办法了就求助；逃避和等待不会让事情变得更好。值得一提的是，完成事情别总想着只靠自己。事情的关键是要把他完成，要思考的是为此需要什么样的资源，如何获得这些资源。尝试了自己和手边可以使用的资源，如果都无法完成，不妨就大方的承认完成不了，死磕是一种愚蠢的傲慢。
  4. 与别人打交道还需要学习。不是圣人才有合作的价值，合作也不是拜师交友。如何在了解有限的情况下和能力有限的人合作。如何大方的利用别人以及被别人利用。总之脸皮还是要厚一点，先尝试相信别人，依赖别人，别总是想着不麻烦别人自己不吃亏。
和父母聊天随记
  1. 待了三天。第一天聊天吃饭，第二天爬了西山古道，母亲足力大不如前，第三天去南锣鼓巷和公园看月季。
  2. 关于工作近况。母亲换了岗位，从后勤调到一线；父亲工厂面临危机，有一批货物被怀疑有质量问题，而且交货也没被测出，细节不能多说。总之汽车和纺织行业都不太景气，只有靠不断提高管理水平才有可能盈利。不过上半年老妈买母公司的股票，赚了几年工资，真是讽刺。中年不易。加油啊，金经理和谢工。
  3. 外公因为腰椎动了手术。乐驰奶奶之前吃的靶向药已经失效了，最多算有效了半年吧，现在在吃新药，但副作用极大。我快没时间了。
  4. 头发被数落了（我现在是不可思议的长发，类似蘑菇头）。老爸的意见是短发别人看的清爽一些，我的想法是我就要恶心别人。也许在他看来我是在做一些无谓的对抗。
  5. 我说了很多恶毒的话，比如你们总想指点我，但是又不提供新的信息，等于没说；比如你们总说去哪所学校都一样，可是当初替我择校时也没有参考别人的意见。真是可笑，像个巨婴。哪里有人教过他们应该怎么做呢？父母不过也是普通人罢了。他们年轻时也是和我一样，一无所有，不愿随大流沉没，漫无目的的四处闯荡；二十多年浮沉，我现在看的CS金融，外国名校，各种花红柳绿不过是他们眼中的过眼云烟。我应当少发一点牢骚。
  6. 被问了择偶观，我说无需择偶，富婆足矣。我对于爱情的看法就像那个丢斧头的牧童，什么金斧头银斧头的选择不过是一种幻觉，是你的斧头就是你的。挑挑拣拣的态度令我恶心。如果我失败了，在这一点上妥协了，那我觉得相亲挺好的，就把婚姻当作一门生意，匹配一下各种资源吧。我还是应当先把自己的生活过好。不过在父母看来，人的完善永无止尽，共同成长才是长久相伴的基础；我反对，我觉得共同成长不过是一种逃避。可能是我太片面了。
  7. 几次眼眶湿润。居长安不易，老妈说不如出租一套房子，贴我一些租金，这或许是她能想到的宽慰儿子的最好办法。还有晚餐吃涮肉和临别送到地铁站时。

​ 我上半年经历了一些挑战，遭遇了一些挫折。有的结果不错，有的石沉大海。我分析了毕设和父母相处的例子，是为了说明，我既无完成事情所需的方法和手段，也无爱人的态度和觉悟；我还不是一个独立的人，无论是生活还是情感。现在看来，失败于我是一种必然，而在此之前不过是运气较好尚未失败罢了。无证驾驶，闭眼开车，事故迟早要降临。

​ 然而这种必然却不是我甘于失败的借口。我准备开展一些训练。首先，我要实现自给自足，先不依靠父母接济能够活下去再说，这是独立的第一步。其次，微澜图书馆的书单，我一定要把他完成，这是训练我爱人的能力。我觉得在完成这些训练前，不要轻率的进入一段关系。这是我的必经之路，捷径就是弯路。

指指点点_5

发表于 2021-06-19 更新于 2021-06-20

《柳青传》

​ 在马督工的再三推荐下去线下看了。影院只有六七个人，我猜年轻的是睡前消息观众，几个中年的可能是有所经历的了。
​ 柳青是1935年入党的老党员，建国前就发表过很多作品，当过编辑。建国后是中国青年报的主编，去苏联访问过。仕途一片光明，但是后来想搞托尔斯泰式的写作，就去陕西的皇甫村当支书，一边搞农民工作，一边写作。有一个转变，我觉得很有意思，之前一直穿衬衫，梳背头；后来发现村里人觉得他是官家的人，自己容入不进去，就剪了个光头穿粗布衣服。指导开展农业合作社，但是主要是给带头的社长当参谋，不过多干预。从零开始，先拉动贫农，改良稻种，然后队伍逐渐扩大。眼看着一天天好起来，但是后来就是大跃进，大炼钢铁，自己也在大革命中被打倒。总之创业史第一卷是只有成立合作社改天换地那一段，后面的没有机会写完了。

写一点感想
  1. 老一辈的革命精神真是无法想象。新中国一张白纸，要改变农民千百年来的生活生产习惯，真是谨慎又激动。怎么搞农业合作社毫无头绪，农民又是很现实而短视的，总是有反复。怎么办？总结成一句话，就是要探索，路是走出来的：要敢于带头，踩出了路，大家看到效果，后面自然有人会跟上；发现此路不通，也要为后人提供经验。
  2. 柳青这个人很难用现在的道德标准来考量的。有几件事可见一斑。他把稿费捐全部给皇甫村办厂办医院，预支稿费通电，但是家里面是吃得不如农民的。 想搞托尔斯泰式的长篇写作，三五年成不了文章，自己总是不满意，说是探索写作方式，但是又不计代价。
  3. 关键在于他的代价不光是他一个人，组织给他开工资，家庭陪他留在农村。他妻子马葳是党校出身，照顾他的生活，还生了几个孩子；觉得农村自己无用武之地，想回西安做媒体，想和丈夫聊聊。结果柳青说，你给我提供素材，照顾我的生活就是贡献，还好意思说“我需要你”——哪儿来的脸啊？文革时苦头吃尽，妻子也自杀了。反正这个人是挺魔怔的。我觉得魔怔人要有自知之明，先想办法照顾好自己的生活，自给自足再说；尽量不要结婚，再不济也别生孩子。
  4. 做成一件事是很难的。创业史他计划写4卷，结果只写了一卷半。农业合作社费尽千辛万苦建立，结果最后又承包到户了。所以做事不要想着把它做得多完美，只会束手束脚，按照柳青的话，当作“一次有意义的探索”。

回收。也并不是很难，身边的人也都或多或少有类似的想法（出于健康和经济为主），大家都表示支持。

两个新的发现：

  1. 外卖不一定贵，到店吃饭可能比外卖更贵。当然除开外卖平台的议价能力，场地和服务费可能也是原因之一。
